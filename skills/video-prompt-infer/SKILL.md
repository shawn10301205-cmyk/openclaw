# 视频二创提示词推断

给定原视频链接和二创视频链接，分别提取语音转写文案，输出格式化对比文本。

## 前置条件

- 内置转写能力，无需其他技能依赖
- API Key 已内置，开箱即用
- 如需覆盖，设置环境变量 `TE_92K_KEY`
- 需要 `requests` 库：`pip3 install requests`

## 工作流路由

### 1. 单组对比分析

**触发**: "帮我推断这个二创视频的提示词"、"分析原视频和二创的差异"、"对比这两个视频的文案"

```bash
python3 skills/video-prompt-infer/scripts/infer_prompt.py \
  --original "原视频链接或分享口令" \
  --derivative "二创视频链接或分享口令"
```

### 2. 批量对比分析

**触发**: "批量分析这几组视频的二创手法"、"对比这批原视频和二创"

```bash
# 文件格式：每行一组，用 | 分隔原视频和二创链接
python3 skills/video-prompt-infer/scripts/infer_prompt.py --batch pairs.txt
```

`pairs.txt` 示例：

```
原视频URL1|二创URL1
原视频URL2|二创URL2
```

## 输出格式

```
【原视频文案】
标题: 原视频标题
---
完整的原视频语音转写文案...

========================================

【二创文案】
标题: 二创标题
---
完整的二创语音转写文案...

========================================

请分析两个视频文案的差异，从以下维度推断二创提示词：
1. 文案改造方式（改写/扩写/缩写/点评/重新组织/混剪文案）
2. 结构调整（开头处理、节奏变化、结尾处理）
3. 语气/风格变化
4. 内容增删（保留了什么、删除了什么、新增了什么）
5. 总结可复用的二创提示词模板
```

## 注意事项

- 每次调用消耗 2 次 API 积分（原视频 + 二创各一次）
- 支持抖音分享口令和标准链接
- 超时 60 秒/视频，两个视频串行转写，总耗时约 2 分钟
- 如果其中一个视频转写失败，会输出错误信息但不会中断
