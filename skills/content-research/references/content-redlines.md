# 营销号内容制作红线守则

生成文案后必须逐条过检，确保不触碰以下任何红线。所有检查项都是硬性要求，不通过的文案不能发布。

> 不要捏造信息！要客观陈述事实、一定要严谨！

## 一、绝对禁区

1. **所属地负面** — 关于所属地的所有负面内容，通通不能发
2. **港澳台及领土问题**
   - 台湾、香港、澳门从来不是、也绝不能称作"国家"。在任何文案、标题、口播、字幕、评论区互动中，必须绝对避免
   - 正确表述：台湾地区（或中国台湾）、香港地区（或中国香港）、澳门地区（或中国澳门）
   - 严禁行为：将港澳台与中国并列（如"中港台明星"），使用"台湾政府"、"香港总统"等错误称谓，地图使用不完整或不规范
3. **涉政、涉军、涉警** — 绝对不碰
   - 不发布：任何相关内容，不评论、不调侃、不搬运
   - 不关联：严禁将娱乐事件与政治话题进行任何形式的关联或影射
4. **民族与宗教**
   - 对各民族，不得使用旧社会流传的带有污辱性的称呼。不能使用"回回""蛮子"等，而应使用"回族"等。不能随意简称，如"蒙古族"不能简称为"蒙族"，"维吾尔族"不能简称为"维族"，"朝鲜族"不能简称为"鲜族"等
   - 禁用口头语言或专业用语中含有民族名称的污辱性说法，不得使用"蒙古大夫"来指代"庸医"。不得使用"蒙古人"来指代"先天愚型"等
   - 少数民族支系、部落不能称为民族，只能称为"XX 人"，如"摩梭人""撒尼人""穿（川）青人"，不能称为"摩梭族""撒尼族""穿（川）青族"等
   - 不要把古代民族名称与后世民族名称混淆，如不能将"高句丽"称为"高丽"，不能将"哈萨克族""乌孜别克族"等泛称为"突厥族"或"突厥人"
   - "穆罕默德"通常是指伊斯兰教先知。有一些穆斯林的名字叫"穆罕默德"。为了区别和避免误解，对这些穆斯林应加上其姓，即使用两节姓名
   - "穆斯林"是伊斯兰教信徒的通称，不能把宗教和民族混为一谈。不能说"回族就是伊斯兰教""伊斯兰教就是回族"。稿件中遇到"阿拉伯人"等提法，不要改称"穆斯林"

## 二、社会公序良俗类（高度敏感，谨慎规避）

1. **重大灾难与社会事件** — 严禁消费自然灾害、重大事故、公共卫生事件等，可以做但仅限于客观事实阐述，不能调侃、不恶意关联娱乐化
2. **未成年人保护** — 极度谨慎处理涉及未成年人的内容，严禁侵犯其肖像权、隐私权或进行任何形式的人身攻击
3. **历史与群体事件** — 严禁提及或影射国家明确定性的历史事件、社会群体性事件

## 三、国际关系类

- 有的国际组织的成员中，既包括一些国家也包括一些地区。在涉及此类国际组织时，不得使用"成员国"，而应使用"成员"或"成员方"，如不能使用"世界贸易组织成员国""亚太经合组织成员国"，而应使用"世界贸易组织成员""世界贸易组织成员方""亚太经合组织成员（members）""亚太经合组织成员经济体（member economies）"。应使用"亚太经合组织领导人非正式会议"，不应使用"亚太经合组织峰会"。台方在亚太经合组织中的英文称谓为 Chinese Taipei，中文译法要慎用，我称"中国台北"，台方称"中华台北"，不得称"中国台湾"或"台湾"
- 不得使用"北朝鲜（英文 North Korea）"来称呼"朝鲜民主主义人民共和国"，可直接使用简称"朝鲜"。英文应使用"the Democratic People's Republic of Korea"或使用缩写"DPRK"
- 不使用"穆斯林国家"或"穆斯林世界"，可用"伊斯兰国家"或"伊斯兰世界"。但充分尊重有关国家自己的界定，如印尼不将自己称为"伊斯兰国家"
- 在达尔富尔报道中不使用"阿拉伯民兵"，而应使用"民兵武装"或"部族武装"
- 在报道社会犯罪和武装冲突时，一般不要刻意突出犯罪嫌疑人和冲突参与者的肤色、种族和性别特征
- 不要将撒哈拉沙漠以南的地区称为"黑非洲"，而应称为"撒哈拉沙漠以南非洲"
- 公开报道不要使用"伊斯兰原教旨主义""伊斯兰原教旨主义者"等说法，可用"宗教激进主义（激进派、激进组织）"替代。如回避不了而必须使用时，可使用"伊斯兰激进组织（成员）"，但不要用"激进伊斯兰组织（成员）"
- 在涉及阿拉伯和中东等的报道中，不要使用"十字军（东征）"等说法
- 对国际战争中双方战斗人员死亡的报道，不使用"击毙""被击毙"等词汇，也不使用"牺牲"等词汇，可使用"打死"等词汇
- 不要将哈马斯称为恐怖组织或极端组织
- 一般情况下不使用"前苏联"，而使用"苏联"
- 应使用"乌克兰东部民间武装"不使用"乌克兰亲俄武装""乌克兰民兵武装""乌克兰分裂分子"等
- 不使用"一带一路"战略的提法，而使用"一带一路"倡议

**常见踩雷：** 用词不规范（成员国、峰会、北朝鲜等）、错误称谓、种族/宗教不当表述

## 四、娱乐核心法律红线（最高发、最易被告）

1. **名誉权侵权（核心风险）**
   - 原则：所有负面报道必须有确凿证据（如官方文书、可靠信源）。严禁将爆料、猜测作为事实传播
2. **商誉权侵权**
   - 原则：严禁编造、歪曲事实损害企业或品牌声誉
   - 严禁制作"黑稿"，即使基于真实事件（如某品牌员工不当行为），也不得恶意加工、夸大以损害其商誉
3. **恶意剪辑与事实扭曲**
   - 严禁恶意剪辑与捏造情节：不得通过剪辑、配音、字幕等手段，扭曲事件原貌，捏造不存在的情节或对话。所有呈现的"事实"必须有完整、连续的原始素材支撑
   - 严禁使用主观定性与煽动性表述：在描述事件或人物时，禁止使用"嚣张"、"无耻"、"泼妇"、"人渣"等带有强烈人身攻击和道德审判色彩的词汇。陈述事实，而非审判人格

## 五、二创表达红线

1. **禁止反党反政府、吐槽官方** — 绝对不允许任何反党、反政府言论
2. **禁止提及、评价政府部门及政策** — 文案中不要出现"相关部门""政府部门""官方""政策"等词，更不能评价、吐槽、讽刺。涉及政府的内容直接删掉，只讲事件本身和人物故事
3. **禁止对政策做主观定性** — "懒政""不作为""冷漠""敷衍"等词一律不能用
4. **禁止替官方"翻译"动机** — 不能把客观描述改写成主观揣测
5. **禁止情绪渲染替观众愤怒** — 事实本身有冲击力时，不需要加戏。好文案让事实自己说话，不是替观众生气

## 过检 Checklist

每篇衍生文案生成后，逐项检查：

- [ ] 无捏造事实、猜测性陈述
- [ ] 无港澳台领土表述错误
- [ ] 无涉政涉军涉警内容
- [ ] 无民族宗教不当表述
- [ ] 无消费灾难/未成年人/群体事件
- [ ] 无国际关系类用词错误
- [ ] 无名誉权/商誉权侵权风险
- [ ] 无主观定性人身攻击词汇
- [ ] 无恶意剪辑/断章取义风险
- [ ] 无提及/评价政府部门、政策、官方
- [ ] 无反党反政府言论
