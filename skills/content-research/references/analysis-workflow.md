# 分析流程与输出格式

## 分析爆款规律

### 找共同点（流量密码）

- 高频标签和情绪词
- 画面风格和核心情绪
- 结构特点（开头方式、叙事节奏）
- 共同点 = 这个赛道的流量密码

### 找不同点（延伸方向）

- 每个爆款的差异化切入点
- 例如：反差冲击、争议讨论、场景迁移、态度金句
- 每个方向 = 一个可延伸的衍生文案方向

## 二创核心原则（必须遵守）

原视频已经是爆款，有被验证的"爆款基因"。二创是换表达方式，不是重新创作：

1. **保留爆款核心** — 原视频的核心观点、钩子、情绪转折必须保留
2. **保留原文钩子和主体** — 原视频的开头钩子和主体叙事段落必须完整保留，不得用模板结构替换
3. **只换外壳不换内核** — 可以换叙事视角、开头方式、句式节奏，但核心论点和素材不能丢
4. **围绕原视频内容展开** — 每篇衍生文案的主题、论据、素材都要来自原视频，不要脱离原视频凭空发挥
5. **口语化不是文章化** — 短视频文案是口语表达，不是公众号文章。禁止"铺垫→递进→升华"的叙事弧线
6. **字数硬性要求** — 每篇衍生文案必须 500-800 字，低于 500 字不合格

## 三阶段递进生成流程（核心）

先学会原视频怎么讲话，再逐步改。不是一上来就套模板重写。

### 阶段一：复刻原视频（复刻度 ~80%）

**目标：** 学会原视频的讲话方式，确认抓住了钩子、主体和情绪。

**做法：**

1. 逐句拆解原视频文案，标注每个部分的功能：
   - 钩子句（开头抓注意力的那句话）
   - 主体段落（核心场景、论据、素材）
   - 情绪转折点（认知反转/情绪升级的地方）
   - 收尾句（最后落在什么上面）
2. 保留以上所有部分的结构和顺序，只做小幅改写：
   - 可以换词但保持原句式节奏
   - 可以删减重复但不能删核心素材
   - 不能加原视频没有的新论点或新场景
3. 输出一篇复刻稿，确认与原视频的情绪密度和口语感一致

**检查点：** 复刻稿读出来，节奏和口语感是否跟原视频接近？钩子是否还在？主体素材是否完整？

### 阶段二：轻度衍生（复刻度 ~50%）

**目标：** 在复刻基础上做适度变化，但核心不变。

**从复刻稿出发，选择一个变化维度：**

- 换开头方式（但保留钩子的冲击力）
- 换叙事视角（第一人称↔第二人称↔第三人称）
- 调整句式节奏（短句为主↔长短交替）
- 增加或精简某个场景的细节

**硬性约束：**

- 原视频的钩子核心意思必须保留（可以换说法，不能换意思）
- 主体叙事段落的核心场景和素材不能丢
- 情绪转折点必须保留，位置可以微调
- 仍然保持口语化，不能变成文章叙事

### 阶段三：深度衍生（复刻度 ~20-30%）

**目标：** 套用模板做方向性改编，但必须锚定原视频的情绪基调和核心素材。

1. 根据延伸方向选择匹配的 prompt 模板
2. **用原视频的钩子和主体素材填充模板**，而不是让模板结构吞掉原文内容
3. 模板只决定"怎么讲"，不决定"讲什么"
4. 生成后对照原文检查：钩子还在吗？主体素材还在吗？还是被模板结构完全替换了？

**选择模板** — 根据每个延伸方向的内容语义，从模板描述表中路由到最匹配的模板文件并读取。不要一次读取所有模板，只读需要的那一个。

## 红线守则审查

**生成文案后必须执行此步骤，不可跳过。**

读取 `references/content-redlines.md` 中的红线守则，逐条审查每篇衍生文案：

1. **逐项过检** — 对照红线守则 checklist 逐条检查每篇文案
2. **标注问题** — 如果文案触碰红线，明确标注哪一条、哪个词/句有问题
3. **修改文案** — 直接修改有问题的内容，不是删除整篇，而是替换违规表述
4. **确认通过** — 修改后再次确认所有检查项通过

审查重点：

- 涉及真实人物：检查是否有捏造事实、主观定性、人身攻击
- 涉及社会事件：检查是否有消费灾难、娱乐化处理
- 涉及地域/民族：检查用词规范
- 涉及国际内容：检查称谓和用词准确性

每篇文案审查后在输出中追加：

```
**红线审查：** ✅ 通过（或列出修改项）
```
